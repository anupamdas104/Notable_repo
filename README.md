# Notable_repo

# Dependencies
    - Python3
    - Flask

# How access/install dependencies
    - activate venv
        - Windows: `> .\venv\Scripts\activate.bat`
        - Linux and MacOS: `$source venv/bin/activate`
    - without venv
        - `> pip install -r requirements.txt`
        - `> python app.py`

# How to run the program
     `> python app.py`
